XYZ_Freq module¶

Frequency calculation module

This program can analyse vibration modes of molecule, visualize the normal mode coordinates, and calculate IR spectrum. Before executing this program, please prepare the chemical structure files in the current directory: <jobname>.xyz is necessary. If you want to visualize normal mode vectors, <jobname>.vesta is also required.

Usage:

python XYZ_Freq.py

>>> Enter the job name: <jobname>
